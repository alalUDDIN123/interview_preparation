# 2.No
 - CSS selectors are patterns used to select and target HTML elements on a web page that we want to apply styling.
   - There are several types of CSS selectors, including:
      - Element selectors: 
         - `p,h1,div`
      - Class selectors: 
         - .head
      - ID selectors: 
         - #head
      - Attribute selectors: 
         - a
      - Pseudo-selectors:
         - :hover
         - :focus
         - :first-child
         - and :last-child.
